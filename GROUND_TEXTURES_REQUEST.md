# Ground Textures - Download Request

Download these textures from Kiln's texture library for terrain materials:

## Required Textures (7 total)

1. **`forest_ground_04`** - Forest floor with leaves and debris (dark, damp)
2. **`ground015`** - Rough dirt for paths (worn, compacted)
3. **`grass001`** - Dead grass patches (brown, patchy, sparse)
4. **`mud004`** - Wet mud for low spots (dark, reflective)
5. **`ground047`** - Gravel for cabin foundation (small stones, gray)
6. **`moss002`** - Moss ground cover (green/brown, soft)
7. **Pine needles** - Orange/brown needle texture (if available, else use `forest_ground_04`)

## Usage

These textures will be applied as Three.js materials to ground planes in the game, not baked into models.

## Export Format

- Export as standard texture files (PNG/JPG)
- Place in `public/assets/textures/ground/`
- PSX aesthetic: low-res, dark, weathered appearance
