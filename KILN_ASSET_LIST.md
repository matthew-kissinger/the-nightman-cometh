## üõ†Ô∏è PRODUCTION WORKFLOW

### For Each Asset:

1. **Create in Kiln**
   - Edit `src/modules/geometry.ts` - Build mesh
   - Edit `src/modules/look.ts` - Apply textures from library
   - Edit `src/modules/metadata.ts` - Set category, name, tags
   - For characters: Edit `src/modules/animation.ts` - Create animation clips

2. **Verify in Kiln**
   - Click "Load Asset" in viewer
   - Check triangle count (stay under budget)
   - Test animations (for characters)
   - Verify textures render correctly

3. **Export GLB**
   - Click "Export GLB"
   - File saved as `{assetName}-v{version}-{seed}.glb`

4. **Import to Game**
   - Move GLB to `the-nightman-cometh/public/assets/models/`
   - Load with GLTFLoader in Three.js
   - Position in scene
